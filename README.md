# Multiplayer Grid Game

## Install Locally

```bash
git clone https://github.com/arjunkumar811/Multiplayer.git
cd Multiplayer
npm install && cd client && npm install && cd ..
```

## Run

Terminal 1: `npm run dev`  
Terminal 2: `cd client && npm run dev`

Open http://localhost:5173
